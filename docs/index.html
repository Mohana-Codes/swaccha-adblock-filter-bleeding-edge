स्वच्छ ब्लिडिङ् एज इडिसन!

is a collection of static AdBlock filters for various popular Nepali sites. It helps you keeps annoying ads away from various popular Nepali sites.

Note: This is not a universal solution to get rid of ads for all popular Nepali sites.

Sites might break often, since its a test-best bleeding edge edition so use it with caution!

Please refer and get back to original स्वच्छ list if you don't like things breaking!
